天天面对各种各样的轮子,有没有想过自己造一个?

看过常见轮子的源码，总是感慨于复杂的设计逻辑。

纸上学来终觉浅，于是乎想换个思路，如果从零开始造（抄）一个轮子，那一定会遇到很多问题，带着问题再去看，也许会容易吧（也许并不容易...）

因为是第一次造轮子，造出来的东西可以想像的到有多粗糙。反正也不能跑那就定位造一个方形的轮子吧。
